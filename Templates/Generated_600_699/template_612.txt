Input: lst: a list of lists
Output: merged: a list of lists where the first and last elements are merged separately

1: Initialize an empty list to store the result, merged
2: Use the zip function with the unpacking operator (*) to pair elements from the same positions across all sublists in lst
3: For each tuple of elements generated by zip:
4:     Convert the tuple to a list
5:     Append the list to merged
6: Return merged
