Input: n: an integer
Output: ans: the number of integers with an odd number of set bits from 0 to n

1: if n is odd:
2:     return (n + 1) / 2
3: count the number of '1's in the binary representation of n
4: ans = n / 2
5: if count of '1's in n is odd:
6:     increment ans by 1
7: return ans
