Input: str1: a string  
Output: str2: a string with even characters removed  
1: Initialize an empty string str2  
2: for each index i in the range from 1 to the length of str1 + 1:  
3:     if i is odd (i % 2 != 0):  
4:         append the character at position i-1 of str1 to str2  
5: return str2
